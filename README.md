# Notas:

Multiplicadora de node por consola que genera txt con la operación.

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Es la base de la multiplicación             [number] [required]
  -l, --listar   Imprime la tabla en consola          [boolean] [default: false]
  -h, --hasta    Define hasta que número se realizara la tabla de multiplicar   
                                                          [number] [default: 10]

```
